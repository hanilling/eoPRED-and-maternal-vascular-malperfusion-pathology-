# eoPRED and maternal vascular malperfusion pathology 
This respository contains code from the paper “A placental DNA methylation score associated with early-onset preeclampsia and maternal vascular malperfusion pathology”. It involves testing for an association between eoPRED score and placental histopathology.


The function to calculate eoPRED score is available in the [PlaNET R package](https://doi.org/doi:10.18129/B9.bioc.planet).
